Path Planner - About





Path Planner is a website designed to help plan a shuttle route where it is required to pick up passengers from different locations. The site uses the genetic algorithm to find the shortest path (by aerial calculation) between all the points. All you have to do is choose the starting, collecting, and ending waypoints and the system will calculate the most efficient route for you.
