How does it work?



Background


Path Planner calculates the shortest path using the genetic algorithm. The system produces generations of trajectories just like generations in our evolution. In each generation there are mutations in the DNA of the entities and there is an evolution of the entity so that it is more suitable to the required conditions, in our case: the lowest number of kilometers.
